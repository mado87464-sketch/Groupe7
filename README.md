# Mini-projet Flask – Gestion des étudiants

## Description
Application Flask simple permettant d’ajouter et d’afficher une liste d’étudiants.

## Technologies utilisées
- Python
- Flask
- SQLAlchemy
- SQLite

## Installation
1. Créer un environnement virtuel
2. Installer les dépendances :
   pip install -r requirements.txt

## Lancement
python app.py
